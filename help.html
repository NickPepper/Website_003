Введите название искомого препарата в поле ввода и нажмите кнопку «ПОИСК». Будет отображена таблица со всеми препаратами, с соответствующими или похожими на введенное Вами название. Для того, чтобы поместить нужный Вам препарат в "корзину", нажмите кнопку с изображением корзины (расположена справа около каждого наименования найденных по запросу препаратов). Заполнив корзину, нажмите кнопку «ОТПРАВИТЬ ЗАКАЗ» - будет выведено отдельное небольшое окно, в котором Вам нужно будет ввести дополнительную информацию (Ф.И.О., контактный адрес и/или телефон) для окончательной обработки Вашего заказа.
